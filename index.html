Elegant New Tab- A better New Tab for Chrome.





Installation

Elegant New Tab is live

Features

  • Inspired by Material Design featuring Color Adaptive Ripple Effect
  • A new background everyday or BYOB (Bring your own background)
  • A new Quote every minute!
  • Weather Data with CSS animations
  • Top Visited Sites
  • Runs Offline
  • Open Source
  • More Features Coming Soon
